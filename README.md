# Auspex Agency OS

Internal operations dashboard for Auspex — tracks client content (posts, reels, stories), social media manager workflow, operations tasks, and graphics deliverables with step-by-step approval tracking.

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Import the repo at vercel.com
3. Vercel auto-detects Vite — click Deploy

Data is saved in the browser's local storage.
